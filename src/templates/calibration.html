Mode d'emploi

  1. Choisissez un fichier vidéo et extrayez une image de référence.
  2. En mode Ligne de comptage : cliquez deux points pour tracer la ligne (les véhicules sont comptés quand ils la traversent).
  3. En mode Zone d'intérêt (ROI) : cliquez plusieurs points pour délimiter la rue. Double-cliquez pour fermer le polygone.
  4. Vérifiez les coordonnées puis cliquez Enregistrer. Redémarrez le conteneur pour appliquer.
